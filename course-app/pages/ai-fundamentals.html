Plain English Request
"Create an Excel formula that finds the highest sale in column B and returns the salesperson name from column A"
Generated Formula:
=INDEX(A:A,MATCH(MAX(B:B),B:B,0))
How it works:
  1. MAX(B:B) finds the highest value in column B
  2. MATCH(...,B:B,0) returns the row number of that value
  3. INDEX(A:A,...) returns the name from column A at that row

Critical Knowledge

What AI Gets Wrong

AI is powerful but not infallible. Understanding its limitations helps you use it safely and catch mistakes before they matter.

Hallucination Example: Fact-Checking Scenario
User Query
"What was Acme Corp's revenue in Q3 2024?"
AI's Response

"Acme Corp reported $847 million in revenue for Q3 2024, representing a 12% year-over-year increase. The growth was driven primarily by their cloud services division."

Confidence: High (but wrong!)
Actual Fact

Acme Corp reported $723 million in revenue for Q3 2024. Growth was 8% YoY, with hardware sales as the main driver.

Source: Q3 2024 Earnings Report (SEC Filing)

Three Key Limitations

Hallucinations AI can confidently state false information
Knowledge Cutoff Training data has a date limit
No True Reasoning Pattern matching, not logical deduction
Golden Rule: Always verify critical information from AI. Use it as a starting point, not a final answer ‚Äî especially for facts, figures, and decisions.
